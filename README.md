## About

This repository contains the source for the-yu-lab website. The site is intended to provide a clear overview of the Principal Investigator's research interests, recent publications, lab members, and ways to get involved.

## Features

- Simple, accessible pages for the PI profile, research areas, and publications
- Contact information and links for collaboration and recruitment
- Clean layout suitable for academic and research group websites

## Getting Started

### Prerequisites

- Node.js (v14+ recommended) and npm, if the site uses a static site generator or tooling
- A code editor (VS Code recommended)

### Local development

1. Clone the repository:

   git clone <repo-url>

2. Install dependencies (if applicable):

   npm install

3. Start the development server or build the site:

   npm run dev
   # or
   npm run build && npm run serve

Adjust commands according to the site tooling in this repository (e.g. Next.js, Gatsby, Hugo, plain HTML).
